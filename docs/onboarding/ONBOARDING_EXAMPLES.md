# Onboarding Examples

## Passive

```yaml
observability:
  serviceName: checkout-api
  environment: dev
  owner: team-commerce
  subscriptionMode: passive
```

## Low-Touch

```yaml
observability:
  serviceName: checkout-api
  environment: staging
  owner: team-commerce
  subscriptionMode: low-touch
```

## Instrumentation

```yaml
observability:
  serviceName: checkout-api
  environment: prod
  owner: team-commerce
  subscriptionMode: instrumentation
```
