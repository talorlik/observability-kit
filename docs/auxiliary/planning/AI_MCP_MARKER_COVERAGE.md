# AI/MCP Marker Coverage

Cross-reference between the AI/MCP layer's `KK-C*` markers and the
contracts, scripts, GitOps artifacts, and runbooks that satisfy each
marker. Cloud-agnostic — every artifact below targets only Kubernetes-
resident components.

The AI/MCP layer is delivered as Batch 14 (see `TASKS.md` and
`TECHNICAL.md` `[TR-15]`).

## Notes

- The smoke wrapper `scripts/ci/validate_batch14_smoke.sh` aggregates all
  KK-C* validators in a single pass.
- New markers should be added to this index when a new AI/MCP capability
  ships.
- The marker IDs map to coverage assertions in each contract's `metadata.marker_coverage`
  field. Search across `contracts/` for `KK-C` to find the canonical
  declarations.
